## Concept
This calculator supports standard arithmetic operations like:
1. Addition
2. Subtraction
3. Multiplication
4. Division
   
The project highlights the following core Qt and C++ concepts:
1. Implementation of the Qt Signal-Slot mechanism
2. Dynamic button to function connections
3. Event-driven programming
4. GUI design using Qt Designer
